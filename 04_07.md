**문제**<br>
0 + 1 + 2 + 3 + .... + 10 = 55<br>
0부터 10까지 더하는 processor를 설계하시오

<br>

**c언어 모델링**
```c
i = 0;
sum = 0;
while(i <= 10){
  sum = sum + i;
  i = i + 1;
outport = sum;
}
halt;

```
